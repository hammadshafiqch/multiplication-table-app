# Multiplication Table Generator

## Overview

The **Multiplication Table Generator** is a web application created using HTML, CSS, and JavaScript. This app allows users to input a number and generates its multiplication table up to 10. With a modern dark theme and responsive design, it provides an intuitive user experience on both desktop and mobile devices.

## Features

- **User Input**: Enter any number to generate its multiplication table.
- **Responsive Design**: Seamlessly adapts to different screen sizes for optimal viewing.
- **Dark Theme**: A sleek dark interface for enhanced readability.
- **Keyboard Support**: Users can generate the table by pressing the "Enter" key or clicking the "Generate Table" button.

## Usage Instructions

1. **Input a Number**: Type a number in the input field.
2. **Generate Table**: Click the "Generate Table" button or press "Enter" to view the multiplication table.
3. **Cross-Platform Compatibility**: The application is optimized for both desktop and mobile devices.
